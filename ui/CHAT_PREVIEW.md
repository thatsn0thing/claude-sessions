## Comparison to Chat Apps

### Similar to:
- **iMessage**: Blue/gray bubbles, timestamps
- **Slack**: Dark theme, technical feel
- **Discord**: Code blocks, system messages

### Different from:
- **ChatGPT**: Less polished, more technical
- **Telegram**: No avatars, simpler styling
- **WhatsApp**: Monospace commands, darker theme

### Unique aspects:
- **Terminal accuracy**: Can toggle to raw view
- **No parsing**: Heuristic-based grouping
- **Command blocks**: Special treatment for shell commands
- **Preserved logs**: Everything is kept
